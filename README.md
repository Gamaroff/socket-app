socket-game
=========

Question and Answer game for two players using Socket.io.

Project uses:

* Knockout.js for MVVM
* Jade for views
* Passport.js for authentication
* Socket.io
* node-uuid for guids
* Twitter Bootstrap for styles
* MongoDB with Mongoose for storage
* bcrypt for hashing passwords
